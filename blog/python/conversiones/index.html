Conversiones

En python hay muchos objetos y estos pueden ser convertidos entre si con algunas excepciones, en esta pagina explico como convertir estos objetos.

int()

Para convertir de string, booleano o float a integer usamos la función int(), su unico parametro es el objeto a convertir, esta función regresa el numero en entero o da error en caso de que no sea posible convertirlo.
Es importante destacar que en el caso de los numeros flotantes estos son redondeados siempre para abajo. 
flotante = 3.7
numero = int(flotante)
print(numero)
cadena = '423'
numero = int(cadena)
numero = numero + 1
print(numero)
booleano = True
numero = int(booleano)
print(numero)
3
424
1

str()

Como aprendimos en la la entrada sobre objetos las cadenas (strings) funcionan distinto que otros objetos cuando se les aplica operaciones matematicas.
Practicamente cualquier objeto puede ser convertido a string como podremos ver en el ejemplo. 
flotante = 1.5
cadena = str(flotante)
cadena = cadena + cadena
print(cadena)
funcion = print
cadena = str(funcion)
print(cadena)
1.51.5
<built-in function print>

float()

Se puede convertir a floatante (float) objetos como cadenas, enteros y booleanos usando la misma sintaxis que los otros convertidores.
Es importante considerar que si se convierte un numero entero, automaticamente se agrega '.0' al final para hacerlo flotante. 
entero = 2
flotante = float(entero)
print(flotante)
texto = '4.5'
flotante = float(texto)
flotante = flotante + 0.7
print(flotante)
booleano = False
flotante = float(booleano)
print(flotante)
2.0
5.2
0.0

bool()

Se pueden convertir numeros flotantes y enteros al igual que cadenas a booleano, es importante destacar en cuales casos el resultado será positivo y en cuales negativo.
En el caso de los numeros, cuando son 0.0 es negativo y en cualquier otro caso (tanto positivo como negativo) el resultado será positivo.
En el caso de las cadenas, si una cadena es completamente vacia el resultado es negativo, en cualquier otro caso es positivo, inclusive si el texto es 'False' el resultado será positivo. 
cadena = 'hola'
booleano = bool(cadena)
print(booleano)
cadena = ''
booleano = bool(cadena)
print(booleano)
numero = -0.1
booleano = bool(numero)
print(booleano)
numero = 0
booleano = bool(numero)
print(booleano)
True
False
True
False

chr()

El objeto char hace referencia a un simbolo, letra o numero en la tabla ASCII, el unico objeto que puede ser convertido es integer el cual representará el codigo en la table del simbolo que necesitamos obtener. 
codigo = 65
caracter = chr(codigo)
print(caracter)
A

list()

Se pueden convertir series (sets), diccionarios (dictionaries), tuplas (tuples) y cadenas (string) a listas, es imporante resaltar que en el caso de los diccionarios solo se mantienen las llaves mientras que los valores son ignorados y en el caso de las cadenas cada simbolo, letra y numero son separados individualmente. 
serie = {5, 2, 6}
lista = list(serie)
print(lista)
diccionario = {'nombre': 'Jeffry', 'pagina': 'jeffser.github.io'}
lista = list(diccionario)
print(lista)
tupla = (3, 6, 2)
lista = list(tupla)
print(lista)
cadena = 'Jeffry'
lista = list(cadena)
print(lista)
[2, 5, 6]
['nombre', 'pagina']
[3, 6, 2]
['J', 'e', 'f', 'f', 'r', 'y']

tuple()

Se pueden convertir series (sets), diccionarios (dictionaries), listas (lists) y cadenas (string), de la misma forma en el caso de los diccionarios los valores son ignorados y se mantienen solo las llaves. 
serie = {1, 7, 2}
tupla = tuple(serie)
print(tupla)
diccionario = {'nombre': 'Jeffry', 'pagina': 'jeffser.github.io'}
tupla = tuple(diccionario)
print(tupla)
lista = ['hola', 'adios']
tupla = tuple(lista)
print(tupla)
cadena = 'Jeffry'
tupla = tuple(cadena)
print(tupla)
(1, 2, 7)
('nombre', 'pagina')
('hola', 'adios')
('J', 'e', 'f', 'f', 'r', 'y')

set()

Se pueden convertir diccionarios (dictionaries), listas (lists), cadenas (string) y tuplas (tuple), en el caso de los diccionarios, los valores se ignoran y se mantienen las llaves.
En el caso de la conversión de cadena (string) y de lista (list) a serie (set), automaticamente es invertido como se observa en el ejemplo y en todos los casos se mantiene la regla de que ningun elemento puede ser repetido, esto debido al comportamiento de los sets, otro comportamiento especial es visto cuando convertimos una tupla (tuple) a serie (set), este será ordenado automaticamente. 
diccionario = {'nombre': 'Jeffry', 'pagina': 'jeffser.github.io'}
serie = set(diccionario)
print(serie)
lista = [5, 8, 8]
serie = set(lista)
print(serie)
cadena = 'Jeffry'
serie = set(cadena)
print(serie)
tupla = (5, 1, 3)
serie = set(tupla)
print(serie)
{'nombre', 'pagina'}
{8, 5}
{'y', 'r', 'f', 'e', 'J'}
{1, 3, 5}

dict()

El caso de la conversión a diccionario es especial, la función convertidora puede tener una cantidad indefinida de parametros, la sintaxis como es mostrada en el ejemplo se basa en (llave = valor), la llave necesita ser predefinida pero el valor puede cambiar según el codigo. 
cadena = 'Jeffry'
numero = 19
booleano = True
diccionario = dict(nombre = cadena, edad = numero, estudiante = booleano)
print(diccionario)
{'nombre': 'Jeffry', 'edad': 19, 'estudiante': True}
