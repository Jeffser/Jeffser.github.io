Objetos

En Python hay muchos tipos de variables, a estos se les conoce como objetos y en este primer post voy a explicar las bases de los objetos en Python.

Integer (int)

El nombre integer significa 'entero' y representa un tipo de objeto en el cual se pueden almacenar numeros enteros tanto positivos como negativos, se pueden realizar diversas operaciones matematicas basicas y completas usando este tipo de objeto, si el resultado no es un numero entero se convierte en el tipo apropiado automaticamente.
Se suele decir que hay limites para los integer, pero desde la llegada de Python3 estos limites fueron removidos. 
numero = 432
print(numero)
numero = numero - 32
print(numero)
432
400

Float

El nombre float significa 'flotante' y representa un tipo de objeto donde se puede almacenar tanto numeros enteros como numeros con decimales en positivo y negativo, aunque cabe resaltar que en el caso de tener un numero entero este será representado con un '.0' al final para denotar que sigue siendo un flotante. 
numero = 12.2
print(numero)
numero = 12
numero = numero - 0.5
print(numero)
numero = numero + 0.5
print(numero)
12.2
11.5
12.0

Char (chr)

El nombre char significa 'caracter' y no es usado a la hora de programar pero es un concepto importante de conocer para usar otros objetos que veremos a continuación, un char es la representación de un codigo de numeros que identifican una letra, simbolo o numero dentro de la tabla ASCII.
En el ejemplo verán el uso de un integer junto a la función chr() para representar la letra 'a', bajo su codigo que es 97. 
codigo = 97
caracter = chr(codigo)
print(caracter)
a

String (str)

El nombre string significa 'cadena' y es uno de los tipos más complejos, aunque por lo general se utiliza para representar tanto letras como palabras y simbolos, es importante saber los usos que tiene, para usarlo tenemos que poner nuestro texto entre comillas dobles o simples.
En el ejemplo pueden observar un string que está encerrado entre comillas dobles pero aún así necesito representar una comilla doble dentro del texto para eso se usa el modificador \ el cual nos ayuda para representar simbolos que normalmente python usaría como la misma barra \.

Boolean (bool)

El nombre boolean se pronuncia en español como 'booleano', es la representación de un bit, este puede ser tanto 0 como 1, o como es representado en Python 'False' y 'True', se usa para representar el resultado de condiciones en distintos escenarios que veremos en los proximos posts.
En el siguiente ejemplo compararemos dos letras y dos numeros usando '==' que significa '... es equivalente que ...', esto deberia de darnos dos posibles resultados, 'True' si ambos lados son equivalentes y en cualquier otro caso 'False'. 
resultado = 'a' == 'b'
print(resultado)
resultado = 1.0 == 1
print(resultado)
resultado = 'A' == 'a'
print(resultado)
False
True
False

List

El nombre list significa 'lista' y representa una serie de objetos en susesión, estos no deben de ser de un tipo en especifico y pueden ser mixtos, inclusive una lista puede tener listas dentro de ella.
Para usarla encerramos entre [] los objetos que queremos en la lista, para agregar más se usa la función .append() y para eliminar .remove() el cual remueve la primera instancia de un elemento en el caso que conozcamos exactamente el objeto o mejor aún .pop() en el caso de que solo conozcamos su indice o lugar en la lista (este valor empieza desde 0). 
lista = [43, "Hola", True, [12, 10], "Hola"]
lista.remove("Hola")
print(lista)
lista.pop(1)
print(lista)
print(lista[2])
lista.append(True)
print(lista)
print(lista[1][0])
[43, True, [12, 10], 'Hola']
[43, [12, 10], 'Hola']
Hola
[43, [12, 10], 'Hola', True]
12

Tuple

El nombre tuple significa 'tupla' y representa una serie de elementos como una lista, aunque esta no es modificable a menos de que se convierta a lista y despues de regreso a tuple.
Para crearla podemos listar todos los elementos dentro de () pero no es necesario, podemos simplemente escribirlos separandolos con una coma. 
tupla = 1, 4, 5, "Hola"
print(tupla)
print(tupla[3])
(1, 4, 5, "Hola")
Hola

Set

El nombre set significa 'serie', como tal este es una serie de items es denuevo como una lista de elementos, la diferencia está con que los items son ordenados automaticamente excepto los strings que son ordenados al azar, en este caso los booleanos tienen el orden False, True; ya que significan 0 y 1 respectivamente, en el caso de los sets es importante resaltar que ningún valor puede estar presente más de una vez.
Para usar un set los items deben de ser encerrados entre {}, no importa el orden en que los pongas, este será ordenado automaticamente. 
serie = {"a", "c", "b", 1, 7, 4, 2, False}
print(serie)
print(serie[3])
{False, 1, 2, 4, 7, 'c', 'b', 'a'}
4

Dictionary (dict)

El nombre dictionary significa 'diccionario' y es una de las formas más utiles de ordenar una serie de elementos, como tal el diccionario es la forma completa de usar un 'set', los diccionarios se dividen en dos partes, llaves y valores, nunca se puede repetir una misma llave en un diccionario, la anterior se verá sobreescrita.
Para usar un diccionario tenemos que encerrar valores entre {} y darles una llave, esto se hace escribiendo en 'string' o 'int' el nombre del indice que queremos seguido por dos puntos (:) y el valor asociado a la llave. 
diccionario = {"nombre": "Jeffr", "color preferido": "naranja"}
print(diccionario)
diccionario['nombre'] = 'Jeffry'
print(diccionario['nombre'])
diccionario['pagina web'] = 'jeffser.github.io'
print(diccionario)
{'nombre': 'Jeffr', 'color preferido': 'naranja'}
Jeffry
{'nombre': 'Jeffry', 'color preferido': 'naranja', 'pagina web': 'jeffser.github.io'}

    return n1 + n2
