Funciones Integradas

Python incluye muchas funciones integradas, hay algunas que funcionan a partir de otros objetos como hay otras que se usan independientes.

print()

Una de las funciones más importantes para el desarrollo de aplicaciones de terminal, print significa imprimir y hace lo que su nombre describe, muestra el objeto que es puesto entre sus parentesis en forma de texto.
Print tiene dos parametros, el primero es el objeto (u objetos) a mostrar y el segundo es el texto a escribir al final, por defecto este es '\n' que significa 'salto de linea', en el ejemplo veremos como puede ser usado para imprimir distintos objetos en una misma linea. 
print(4, 'hola', True)
print('hola', end='-')
print('adios')
4holaTrue
hola-adios

input()

Otra función importante para el desarrollo de aplicaciones de terminal es input(), significa entrada y hace lo que el nombre sugiere, espera por la entrada del usuario y la regresa al codigo. Es muy importante destacar que en todos los casos sin importar que escriba el usuario siempre se va a regresar una cadena (string), si queremos un numero tendremos que convertir ese string.
Input tiene un parametro opcional, si incluimos un objeto este será impreso antes de la entrada del usuario, en el ejemplo el usuario escribió 'Jeffry', 2 y 4. 
nombre = input('Cual es tu nombre? ')
print('Hola ', nombre, '!')
num1 = float(input('Primer numero a sumar: '))
num2 = float(input('Segundo numero a sumar: '))
print('El resultado es ', num1 + num2)
Cual es tu nombre? Jeffry
Hola Jeffry!
Primer numero a sumar: 2
Segundo numero a sumar: 4
El resultado es 6.0

range()

La función range() regresa una tupla que representa un rango, tiene un parametro obligatorio y 2 opcionales, el obligatorio define hasta donde llega el rango, el primero opcional define desde donde empieza el rango, por defecto ese valor es 0 y el segundo opcional define de cuanto en cuanto va a subir el rango por defecto es uno.
Por lo general range() es usado en for aunque no es la unica forma de usar un for, en el ejemplo mostramos como actua según los parametros y como se comporta en un for. 
rango = range(5)
print(rango)
rango = range(2, 10)
print(rango)
rango = range(5, 20, 2)
print(rango)
for i in range(3, 20, 2):
    print(i, end="-")
range(0, 5)
range(2, 10)
range(5, 20, 2)

type()

La función type nos regresa el tipo de objeto que ponemos como parametro, es util por ejemplo para saber si un numero en un objeto funcionará como float, int o string y saber si se tiene que convertir. 
cadena = '23'
tipo = type(cadena)
print(tipo)
entero = 23
tipo = type(entero)
print(tipo)
comparacion = tipo == int
print(comparacion)
<class 'str'>
<class 'int'>
True
