Instances

Instances are managers of all types of Models, specifically Added Model Instances can be modified inside the Instance Manager.

Instance Manager

To access the instance manager the user can use the option in the Main Drop-Down Menu, or press ctrl+i from the Main Window .

From here the user can manage all the existing Added Model Instances. The available preferences vary depending on the provider.

There can only be one active instance at a time, to select an instance simply click it

Instance Preferences

Each instance has different properties.

Added Model Instances

These instances control all the Added Models, these instances can either be online or local depending on the provider.

Ollama Instances

Ollama is the main focus of Alpaca, it is the most tested and has the most features (like Creating Custom Models).

To consult the list of available models visit Ollama Library

Alpaca provides different types for this instance but they all operate in mostly the same way.

Ollama (Managed)

This instance becomes available when Alpaca auto-detects an installation of Ollama (or when using the Flatpak Extension), when using it, Alpaca will run Ollama by itself, making sure the processes are only active when Alpaca needs them, additionally, the user can select a model directory and set overrides to troubleshoot GPU issues.

This is the recommended instance to use with Alpaca.

Ollama (External)

This instance connects to an existing Ollama instance, in this case Alpaca won't manage it, although it mostly works the same way.

A recommended setup to use with Alpaca is running Ollama in a Docker Container and connecting it to Alpaca through this instance.

Ollama (Cloud)

Recently Ollama started providing cloud services through their own server, this instance is fully online and managed by Ollama.

To use it an API key from an Ollama Account is required.

OpenAI Instances

Every other instance is considered an OpenAI Instace this is because they rely on the OpenAI Python library to operate.

Most of these instance have not been tested properly because of the need to buy tokens, their support is based on the available documentation on how they connect to OpenAI's library.

Text to Speech Instance

Managed locally by Kokoro, it provides small voice models that can be used to dictate Messages, Generating Podcasts, etc.

Alpaca does not depend on Kokoro which means that some unofficial packages might choose not to include it, in those cases make sure Kokoro is installed on the system, Alpaca should detect it and show the options that depend on it.

The official Alpaca Flatpak includes Kokoro, in that case the user won't have to do anything to make it work.

For a list of voices consult the Kokoro Documentation, please note that the official Alpaca Flatpak currently doesn't include Japanese and Mandarin Chinese voices due to them needing additional dependencies.

Speech to Text Instance

Managed locally by OpenAI Whisper, it provides models that can detect speech from audio coming from files or directly from a microphone, it is used in the Transcriber Activity and the microphone inside the Global Footer's Action Bar.

Alpaca does not depend on OpenAI Whisper which means that some unofficial packages might choose not to include it, in those cases make sure OpenAI Whisper is installed on the system, Alpaca should detect it and show the options that depend on it.

The official Alpaca Flatpak includes OpenAI Whisper, in that case the user won't have to do anything to make it work.

Available models:

  • Tiny (~76 MB)
  • Base (~151 MB)
  • Small (~488 MB)
  • Medium (~1.5 GB)
  • Large (~2.9 GB)

For the list of available languages, please consult the OpenAI Documentation

Background Remover Instance

Managed locally by RemBG, it provides models that can remove backgrounds from images easily, it is used in the Background Remover Activity.

Alpaca does not depend on RemBG which means that some unofficial packages might choose not to include it, in those cases make sure RemBG is installed on the system, Alpaca should detect it and show the options that depend on it.

The official Alpaca Flatpak includes RemBG, in that case the user won't have to do anything to make it work.

Available models:

  • U2Net Light (~5 MB)
  • U2Net Human Trained (~168 MB)
  • Silueta (~45 MB)
  • Isnet General Use (~170 MB)
  • Isnet Anime (~170 MB)
