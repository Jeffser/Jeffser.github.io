Alpaca

Chat With AI Models

Have conversations with AI models locally without any data collection outside your device. Asks questions about advanced topics, use files like PDF and even images to ask questions.

All without any complicated setup, just install Alpaca, download a model of your choice and start chatting!

Privacy

Alpaca is powered by Ollama, an open source framework for running AI Models, mainly developed by Meta (Formally Facebook), Ollama does not collect any information about conversations or usage of models.

The model list used by Alpaca is provided by Ollama, models have no direct access to the internet, the information collected is stored locally on your device.

  • Alpaca does not collect information outside your device
  • All conversations are stored locally using JSON files
  • The only times Alpaca connects to the internet are to extract text from websites, extract YouTube captions and download models
  • You don't need internet access to use Alpaca with your already downloaded models
  • There's no account system or cloud computing involved on the usage of Alpaca or Ollama
  • The quality and speed of responses is determined by your system specifications, the model used and nothing more
  • These disclaimers may not apply if you use a remote instance of Ollama (optional)

Local AI Models

Experiment with dozens of pre-trained models from companies like Meta, Microsoft, Google, Nvidia and Alibaba, as well as independent models available through a simple interface.

From coding, multilingual, image recognition and specialized models, Alpaca can combine models in a single conversation so you can always find the answer to any question.

Image Recognition

Ask image recognition capable models questions about any image, this process is also fully offline and private, your images and text will always stay between you and the model.

You can also use text files, PDF, YouTube videos and even websites to prompt a model!

YouTube Recognition

By extracting the captions from a YouTube video automatically, you can attach music, tutorials and any kind of video to ask questions to a model.

Just paste the url, select a language and Alpaca will take care of everything.

Disclaimer: Please do not use this feature to plagiarize content from other users, please support the creators of the content you are using by watching their videos with ads or YouTube Premium.

Code Generation

Most models are capable of generating code, Alpaca automatically transforms the code into code blocks to make it easier to copy and review the code

Additionally, you can also run Python scripts directly from Alpaca using a virtual environment that supports features such as user input and library installation.
